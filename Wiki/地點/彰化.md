# 彰化
**類型**：縣市 / 地點
**國家**：台灣
**上層地點**：台灣中部（彰化縣）
**相關地點**：[[台中]]、[[雲林]]、[[南投]]、[[台灣]]

## 特色
彰化（Changhua）是台灣中部縣份，緊鄰台中市南側，以彰化員林一帶為主要城市，農業資源豐富，八卦山為代表景觀。成人娛樂方面，彰化的消費水準比台中略低，是中彰投地區的次要消費據點，並以個工（個體戶）形式的戶外/車震服務聞名於節目中。

**在節目中的脈絡（S3EP52，嘉賓南屯里里人）**：
南屯里里人分享大學時期在彰化深山約個工的荒誕體驗——對方不設工作室，主打**野外/車震**，在半夜騎摩托車出現在彰化深山。

**彰化深山個工事件（S3EP52）**：
- 南屯里里人大學時透過某管道約到彰化山區個工
- 妹子沒有固定工作室，**主打野外和車震**
- 約定地點：彰化深山山路
- 妹子騎著摩托車在**半夜**出現
- 服務內容：車內口爆服務（口技）
- 服務完後：騎車離開，全程保密高度高
- 評價：過程極具刺激感，荒誕而獨特

**在節目中的脈絡（S3EP255，聽眾投稿）**：
聽眾分享曾認識一名越南妹子（神似李珠恩），後來得知該妹子在台灣彰化越南店工作，是跨城市跨地域的服務人員流動案例。

## 師台越南店攻略（S3EP264，[[大口袋]]介紹）

彰化有「師台制」越南妹店，玩法與台北、台中不同：

| 項目 | 內容 |
|------|------|
| 計費 | 300 TWD/小時，全歸妹子（老闆賺包廂費和酒錢） |
| 最低消費 | 兩小時 600 TWD 起（不含酒和秀舞） |
| 師台 | 固定不轉檯，一個妹子從頭陪到尾 |
| 秀舞 | 每首歌 200 TWD，客人可自己點舞曲觸發 |
| 妹子組成 | 外配姐姐（30-40 歲，老公上班打工賺外快）+ 越南留學生（政府新南向政策招來）|
| 姐姐尺度 | 較大，可手工（行情 1,000 TWD） |
| 年輕妹 | 保守，看緣分，但可以待到隔天 |

**第一次去的攻略：**
1. 老闆只能安排條件一般的姐姐（紅牌年輕妹自己排客已排滿）
2. 請姐姐把年輕妹叫進包廂「訪客」
3. 給年輕妹小費，當場要 LINE / 聯絡方式
4. 下次直接找年輕妹私約，不走老闆管道

**靠行制：**
- 小姐全部自己排客，是靠行性質
- 老闆提供場地，妹子自己決定接哪些客人
- 人氣年輕妹往往約不到，要直接私聯
