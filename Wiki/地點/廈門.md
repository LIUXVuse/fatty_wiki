# 廈門
**類型**：城市 / 地點
**國家**：中國
**上層地點**：福建省（Xiamen, Fujian）
**相關地點**：[[福建]]、[[廣州]]、[[深圳]]、[[中國]]

## 特色
廈門（Xiamen）是福建省東南部的港口城市，以**鼓浪嶼、溫暖氣候、閩南文化**著稱，是台商台灣人常去的大陸城市之一（語言相近，閩南語可溝通）。成人娛樂方面，廈門以**個工（個體戶）**為主要形式，多透過 Telegram 群組媒合，費用約 600 RMB，但照騙和代聊問題嚴重。

**在節目中的脈絡（S3EP179，嘉賓老張）**：
老張出差廈門期間透過 Telegram 找個工，連續遭遇照騙（代聊），分享辨識真人個工的完整攻略。

**廈門個工現況（S3EP179）**：

| 項目 | 詳細 |
|------|------|
| **費用** | 約 600 RMB |
| **媒合管道** | Telegram 群組/頻道 |
| **主要問題** | 照騙、代聊比例極高 |
| **時間管理** | 服務人員接客密集，限時約 40 分鐘（含洗澡），節奏極快 |
| **特色** | 個工型，非店面，在旅館完成服務 |

**Telegram 照騙驚魂事件（S3EP179，老張親身遭遇）**：
1. **第一次**：被照片吸引，對方要求先匯 100 RMB 車馬費，老張照付。飯店大廳出現的是一位年紀相仿的大媽（大神）。老張在電梯口直接打槍趕走。
2. **第二次**：透過短視頻與刺青比對確認本人，成功見面。但服務過程中對方全程用手機回覆其他客人，限時 40 分鐘，快到時催促「快點射否則要下課」。

**TG 找廈門個工避坑指南（S3EP179）**：

| 辨識方法 | 說明 |
|---------|------|
| **短視頻驗真** | 不能只看照片，必須找有「露臉短視頻」的帳號 |
| **刺青交叉比對** | 比對視頻與照片中的刺青位置、臉型特徵是否吻合 |
| **評價篩選** | 個人 Channel 評價需超過 10 條且有好有壞（純好評反而可疑）|
| **罐頭訊息警示** | 多個群組同一帳號發相同內容 → 必為罐頭帳號 |
| **車馬費警示** | 要求先匯款車馬費 → 風險極高，極可能是照騙 |

**廈門 vs 廣州成人娛樂比較（S3EP179）**：

| 城市 | 主要形式 | 費用 |
|------|---------|------|
| **廈門** | 個工（TG 媒合）| 約 600 RMB |
| **廣州** | KTV 座台 | 座台 1200 RMB / 短中 3000+ RMB |
| **越南（替代）**| KTV / 按摩 | 更透明、性價比高 |

**廈門個工工作節奏描述（S3EP179）**：
> 「進房時剛好有另一個嫖客走出，出房時又有另一個嫖客進來。服務過程中還在用手機回覆其他客人，快到 40 分鐘時催促趕快射。」

**廈門沉船背景（S3EP10，嘉賓方丈）**：
方丈出差廈門時帶了一名大學輟學的網路交友妹子同行，同時在廈門另有長期聯繫的對象，因連續三天未回訊息導致被抓包。
