# 基隆
**類型**：城市 / 地點
**國家**：台灣
**上層地點**：基隆市
**相關地點**：[[台北]]、[[台灣]]

## 特色
基隆（Keelung）是台灣北部的重要港口城市，以廟口夜市、海鮮聞名。成人娛樂方面，基隆最著名的是**龍安街「鐵支路」**（鐵之路）豆干厝（私娼），以快速消費、平民化定價著稱。位於平交道（鐵路平交道）旁的舊平房建築群，是台灣傳統私娼街的代表性場所之一。

**在節目中的脈絡（多集）**：
節目多次提及基隆作為大台北周邊的傳統豆干厝消費據點，「鐵支路」為主要介紹對象。

## 龍安街「鐵支路」豆干厝

| 項目 | 詳細 |
|------|------|
| **地點** | 基隆龍安街，平交道旁舊平房 |
| **費用** | 約 1,000-2,000 TWD |
| **建築特色** | 平交道旁舊式平房，傳統私娼建築型態 |
| **最佳造訪時間** | 週五或週六傍晚六點後 |
| **服務特徵** | 快速消費，直接明瞭 |

**「鐵支路」特色**：
- 舊式平房建築（豆干厝型態），歷史悠久
- 位於鐵路平交道旁，地標明顯
- 以周末傍晚後客流最旺
