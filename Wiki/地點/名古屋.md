# 名古屋
**類型**：城市 / 地點
**國家**：日本
**上層地點**：愛知縣（Nagoya, Aichi）
**相關地點**：[[日本]]、[[東京]]、[[大阪]]、[[飛田新地]]

## 特色
名古屋（Nagoya）是日本愛知縣首府，日本第三大都市圈，以豐田汽車總部、名古屋城、味噌豬排著稱。成人娛樂方面，名古屋有**泡泡浴（Soapland）**和**半套店**，其中部分高端店藏在住宅區，外觀像普通民宅，內部裝潢卻極其豪華，是日本風俗業「隱密豪華」風格的代表。

**在節目中的脈絡（S3EP70，嘉賓可樂）**：
可樂與同事出差名古屋，透過**風俗雜誌折價券**，在住宅區找到一間外觀像民宅、內部金碧輝煌的泡泡浴，體驗日式服務的極致 SOP。

**在節目中的脈絡（S3EP136，投稿者廣州台琴）**：
分享名古屋某半套店的體驗，稱服務「屌打台灣」，妹子英文多益 800+ 水準，有正規按摩證照，並提供前列腺保養服務。

**名古屋泡泡浴體驗（S3EP70，可樂）**：

| 項目 | 詳細 |
|------|------|
| **費用** | 20,000 日幣（約 5,000 TWD）|
| **外觀** | 住宅區民宅外表，絕對猜不到是風俗店 |
| **內部** | 金碧輝煌，水床設備，豪華裝潢 |
| **服務流程** | 幫客人脫衣 → 整齊摺好衣服 → 協助入浴 → 水床磨蹭服務 |
| **服務精神** | 日式極致細心，每個環節都有 SOP |
| **折扣** | 透過**風俗雜誌折價券**可減價（雜誌在便利商店可買）|

**名古屋半套店（S3EP136）**：
- **費用**：較貴（具體未揭露）
- **妹子**：英文程度極佳（多益 800+ 水準），擁有按摩正規證照
- **服務**：提供前列腺保養（特殊項目），技術專業
- **選項**：可追加「比基尼選項」，且床側有鏡子供視覺享受
- **評價**：服務屌打台灣，無論溝通還是技術都遠超一般

**可樂分享的日本情色 SOP（S3EP70）**：
> 晚餐 → Snack Bar（日式酒店，僅聊天喝酒，不觸摸）→ 開雜貨（前往泡泡浴/半套店）

- **Snack Bar**：純喝酒聊天的日式酒吧，老闆娘和媽媽桑陪客，無肉體服務
- **開雜貨**：進入實際性消費環節的俚語
- **風俗雜誌**：日本便利商店可買的成人店廣告雜誌，附有折價券

**名古屋早班優勢（S3EP70 多位投稿者）**：
1. **衛生**：避開前一客人服務後的殘留
2. **體力**：妹子體力充沛，女上位更積極
3. **態度**：未遇爛客，精神好，服務態度佳
4. **安全**：警方早晨臨檢機率低

**飛田新地比較（S3EP70 可樂提及）**：
- 大阪飛田新地：傳統日本紅燈區，但「價格昂貴且過程太快」
- 名古屋泡泡浴：CP 值更高，服務時間長且細膩
