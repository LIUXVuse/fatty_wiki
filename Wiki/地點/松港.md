# 松港
**類型**：行政區 / 地點
**國家**：中國
**上層地點**：[[深圳]]（廣東省）
**相關地點**：[[深圳]]、[[東莞]]、[[廣東]]、[[中國]]

## 特色
松港（Songgang，松崗）是深圳寶安區的一個行政區域，屬深圳西北部工業重鎮，以製造業和外來務工人員密集著稱。松港一帶有按摩店聚集，但受中國「嚴打」政策影響，許多店家採取**外觀誘導但實際純按摩**的策略——妹子穿著性感但只提供正規按摩服務。

**在節目中的脈絡（S3EP145，嘉賓老張）**：
老張出差深圳期間前往松港按摩店，遭遇「誘導式裝潢＋純按摩」的避坑場景，並提出辨識技巧。

**松港按摩店現況（S3EP145）**：

| 項目 | 詳細 |
|------|------|
| **代表店家** | 未具名按摩店（松港） |
| **費用** | 388 RMB / 小時 |
| **妹子穿著** | 短裙、低胸上衣，外觀誘導性強 |
| **實際服務** | 僅純按摩，無特殊服務 |
| **陷阱** | 光看穿著以為有特殊服務，實際上沒有 |

**避坑重點（S3EP145 老張親身踩雷）**：
> 「不能僅憑妹子的穿著判斷是否有特殊服務，要直接詢問幹部（管理者）明確的服務項目。」

- 目前大陸按摩店採取「模糊報價」策略：外觀吸引客人進來，但實際服務保守
- **正確做法**：進店後直接問幹部（非妹子本人）：「有沒有特別的服務？」
- 若幹部支吾其詞或說「看妹子」→ 通常代表無特殊服務，謹慎決定是否消費

**深圳成人娛樂現況背景**：
- 深圳受廣東省「嚴打」政策影響，2018 年後整體規模大幅收縮
- 松港、寶安等工業區仍有少量按摩店存在，但服務透明度低
- 高端服務（攝護腺、龍筋）多集中在羅湖、南山等中心城區
- **老張建議**：若追求明碼標價與服務透明，更推薦前往**越南**而非留在大陸
