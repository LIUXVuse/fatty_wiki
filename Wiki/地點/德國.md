# 德國

**類型**：國家
**主要城市**：[[法蘭克福]]、[[柏林]]、[[慕尼黑]]

## 特色
德國是歐洲成人娛樂最合法化的國家之一，以 **[[FKK]]（天體俱樂部）**聞名。法蘭克福郊區 [[Oasis]]、法蘭克福市區 [[Shark]]、柏林 [[Artemis]] 是節目最常推薦的三大店。**建議選擇法蘭克福等大城市**，小城市可能對亞洲人冷淡或有種族歧視（S3EP91）。

## FKK 流程（S3EP81）
1. 付入場費 → 2. 置物櫃存放貴重物品 → 3. 洗澡 → 4. 換浴袍 → 5. 在場內自由選妹 → 6. 談好服務項目與費用 → 7. 完成後至櫃台結帳

**FKK 實戰三寶**：充足現金 + 壯陽藥物 + 香菸
