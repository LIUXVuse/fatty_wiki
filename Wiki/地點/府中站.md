# 府中站
**類型**：捷運站 / 商圈 / 地點
**國家**：台灣
**上層地點**：新北市板橋區
**相關地點**：[[板橋]]、[[新北]]、[[台灣]]、[[林森北]]

## 特色
府中站（Fuzhong Station）是台北捷運板南線在板橋區的車站，周邊是板橋最重要的商業核心之一，有遠東百貨、南門街旅館區等，交通方便且好停車。在節目中，府中站被推薦為**包養網約會地點**的首選咖啡廳（星巴克）的所在地，同時也提及該區附近曾有長平時期（東莞全盛時期）知名的藝豪、君悅、美寶等店。

**在節目中的脈絡（S3EP64，嘉賓 Cam）**：
Cam 分享包養網實戰 SOP，將**府中站星巴克**列為台北/板橋地區包養網初次見面的首選地點，理由是：交通方便、好停車、附近有旅館（南門街）和商場（遠東百貨）。

**府中站作為約會地點（S3EP64）**：

| 優點 | 詳細 |
|------|------|
| **交通便利** | 捷運直達，無需開車就能到 |
| **停車方便** | 板橋停車場多，不怕找不到車位 |
| **後續延伸** | 附近南門街有旅館，確認發展後步行可達 |
| **環境得體** | 星巴克作為首見地點，有品但不奢靡 |

**包養網約會 SOP（S3EP64 Cam 完整分享）**：

| 步驟 | 說明 |
|------|------|
| **開場** | 讚美對方外型、氣質或特定部位（眼睛、嘴巴），絕不直接開價 |
| **Probing** | 詢問是否有配合經驗，讓對方主導提出收費模式 |
| **關鍵詞** | 表示在找「長期對象」（即使只想約一次），成功率較高 |
| **篩選** | 交換 LINE、照片，確認外貌後再約見面 |
| **首見地點** | 府中站星巴克（板橋）/ 潮附路星巴克（台中）/ 金品城星巴克（新竹）|
| **見面準備** | 穿著得體、備小禮物（護手霜、耳機），車馬費 1500 上下 |

**避坑指南（S3EP64）**：
- 過於專業的網美照（可 Google 以圖搜圖驗真偽）→ 通常是假圖
- 臉部打馬賽克且簡介只有「可甜可鹹」→ 通常是業者帳號
- 第一次見面就要求開房 → 查莊（設局）
- 一發訊息就直接留 LINE → 通常是八大業者
- 體重通常少報：標榜「微胖」要心理預期加 10-15 公斤

**林森北路踩雷警示（S3EP64 同集）**：
投稿者魯邦 Mike 在林森北路一發未射的三連踩雷教訓 → 中山區消費建議做足功課再行動。

**長平時期舊店（東莞，非台灣）補充**：
節目中提及「藝豪、君悅、美寶」均為東莞長平時期（全盛年代）的 KTV 店名，並非位於台灣府中站附近，僅為同集中提及：
- **藝豪**：長平時期水準不錯的 K 房
- **君悅**：長平時期推薦桑拿店
- **美寶**：妹子較多，有知名「豆花妹」，需提前 1-2 天預約

**台灣各地包養網約會首見地點推薦**：

| 城市 | 推薦首見地點 | 原因 |
|------|-----------|------|
| **板橋（台北）** | 府中站星巴克 | 交通方便，附近有旅館（南門街）|
| **台中** | 潮附路星巴克 | 好停車，附近有汽車旅館與老虎城 |
| **新竹** | 金品城星巴克 | 位於火車站對面，附近有護城河景觀 |
