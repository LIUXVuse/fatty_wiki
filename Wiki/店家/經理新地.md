# 經理新地

**類型**：日本風俗店（接客需日文溝通）
**地點**：[[日本]][[大阪]]
**服務類型**：日式性服務（非外國人友善）

## 特色

**大阪風俗店，不推薦，需自行與媽媽溝通，看不到人，不會日文者請跳過**（S3EP238，嘉賓老低）：
- 運作模式：顧客需**自行與媽媽桑（老闆娘）溝通**，事先談好人選與費用
- **進店前看不到妹子**，無法直接現場選妹
- **日文門檻高**：不會日文的外國人幾乎無法使用
- 性價比評價差，老低主持直接說「不推薦」

## 大阪風俗店外國人接客等級（S3EP238）

| 等級 | 條件 |
|------|------|
| 1（最友善）| 全部接，含無日文/無預約 |
| 2 | 原則不接→有機預約可→不看日文 |
| 3 | 原則不接→有機預約可→需日文面試 |
| 4 | 接→需具備日文程度 |
| 5 | 接→部分妹子接 |
| 6 | 全部接 |

> **經理新地 = 等級 3–4**，需日文溝通，不適合非日文母語者
