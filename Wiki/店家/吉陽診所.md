# 吉陽診所

**類型**：泌尿科診所（聽眾友善）
**地點**：[[台灣]][[台中]]
**服務類型**：泌尿科門診、性病治療、壯陽藥、PrEP、HPV 疫苗

## 特色

聽眾友善診所。告知是「肥宅老司機」節目聽眾，可心照不宣討論特殊性行為或保養需求，不會被評判。主治醫師蕭醫師，可要求 One on One 獨立診間保障隱私。

| 項目 | 備註 |

| 犀利士 5mg | 壯陽藥 |
| PrEP 預防投藥 | HIV 暴露前預防 |
| HPV 疫苗 | 性病預防 |
| 性病治療 | 泌尿科門診 |

## 實用技巧
- 暗號：「肥宅老司機」，醫師心照不宣
- 可要求 One on One 診間（蕭醫師，S3EP238）
