# 義豪

**類型**：桑拿（成人娛樂）
**地點**：[[中國]][[東莞]][[長平]]（原長坪）
**服務類型**：桑拿按摩（含特殊服務）

## 特色

**東莞長平水準較高的桑拿，提供招牌「火燒鳥」服務**（S3EP35，嘉賓老張、Ken）：
- 長平（長坪）地區水準較高的桑拿店之一
- 招牌服務：**「火燒鳥」** — 用生薑與酒精噴灑於生殖器後**點火**，名義上是「除濕」（特殊刺激體驗）
- 老張建議路線：**下午去樓上桑拿，晚上二樓逛花街**

## 長平桑拿行程路徑（S3EP35）

```
桑拿（洗澡/修幹）→ K 房（喝酒/挑妹）→ 房間（正式服務）
```

| 步驟 | 說明 |
|------|------|
| **下午** | 到義豪樓上桑拿，享受修幹服務 |
| **晚上** | 二樓逛花街，挑選 K 房妹子 |
| **循環** | 此行程可使人數日不出酒店 |

## 現況（S3EP35，S3EP62）

- **東莞長平成人娛樂現已大幅沒落**，因 2014 年央視報導引發大規模掃黃
- 「東莞站」因名字太直白改名「長平站」
- 義豪等多數長平桑拿已關閉或大幅縮水
- 掃黃後部分玩家轉移至[[惠州]][[翡翠]]等地，但評價不佳
