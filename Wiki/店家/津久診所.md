# 津久診所

**類型**：診所（非成人場所）
**地點**：[[台灣]]
**服務類型**：男性性功能保養、性病預防

> [!warning] ⚠️ 類型更正
> 此頁原被標記為「成人娛樂」，實為醫療診所，提供合法的醫療服務。

## 特色
S3EP258 的來賓診所，由兩位醫師上節目分享男性性功能保養與性病預防知識。提供震波治療、PrEP/Doxy 預防與 HPV 疫苗接種，支援**全匿名就診**。

## 服務項目
| 項目 | 說明 |
| :--- | :--- |
| 震波治療 | 刺激微血管再生，改善血管性勃起功能障礙，效果比藥物更持久 |
| PrEP | 暴露前預防，降低 HIV 感染率；現有注射劑可取代每日口服 |
| Doxy | 預防梅毒、衣原體等細菌性性病 |
| HPV 疫苗 | 預防菜花、頭頸癌、肛門癌 |

## 就診技巧
- 暗號「**皮卡**」→ 可安排匿名就診，避開女護理師，適合不好意思的患者
- 支援全匿名，適合因害羞而延誤就醫的人
