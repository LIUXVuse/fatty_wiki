# 金沖

**類型**：禮服店（酒店）
**地點**：[[台北]]，[[台灣]]
**服務類型**：陪酒秀舞

## 特色
台北的禮服秀場酒店，在節目中評價極差。主持人親身體驗後留下深刻（負面）印象：

> 妹子品質如「康瓦斯」（平庸僵硬），關燈秀舞後發現是兩位上半身赤裸且身材壯碩的女性，視覺衝擊極大，主持人全程未碰妹子，僅靠聊天度日。

## 評價
| 項目 | 評價 |
| :--- | :--- |
| 妹子顏值 | 低，形容為「康瓦斯」等級 |
| 身材 | 部分壯碩 |
| 例外 | 僅有幹部（管理人員）較正 |
| 總評 | ❌ 不推薦 |

## 注意事項
- 若只是為了喝酒聊天，尚可接受
- 不適合對外貌有要求的玩家
