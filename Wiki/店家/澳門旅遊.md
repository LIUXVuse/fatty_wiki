# 澳門旅遊（克里斯直播主伴遊事件）

> [!warning] ⚠️ 類型更正
> 此頁記錄的並非實體店家，而是一次以金主身份帶直播主伴遊的旅遊行為。歸類為「案例 / 攻略事件」較為準確。

**類型**：伴遊案例（非實體店家）
**地點**：[[澳門]]
**服務類型**：金主伴遊（直播主約出）

## 事件概述
聽眾克里斯（S3EP11）透過在 [[17直播]] 平台大量斗內，成功約出一名外貌出眾的網紅直播主，並帶其前往澳門三天兩夜旅遊。整個過程涉及金錢攻勢與心理博弈，是節目中討論「直播主攻略戰」的經典案例。

## 費用結構
| 項目 | 費用 | 備註 |

| 前置打賞（17直播） | 約 20 萬 TWD | 引起注意、建立金主身份 |
| 澳門機酒食 | 約 10 萬 TWD | 三天兩夜全包 |
| 零用錢 | 1 萬 HKD | 交給直播主的個人零用 |
| **總成本（估算）** | **約 33 萬 TWD** | |

## 攻略流程（克里斯版）
1. **篩選目標**：觀察 IG/FB 生活奢侈程度遠超公開職業收入 → 有「搞頭」
2. **建立金主身份**：在 17直播 斗內 20 萬點，對方提供私人 LINE
3. **心理拉扯**：對方拿喬拒絕約見 → 故意轉去競爭對手直播間消費
4. **損失厭惡觸發**：直播間「騎士團」通報後，對方主動約咖啡
5. **成功約出**：帶往澳門旅遊，達成目的

## 實用知識（節目整理）
- **直播主篩選法**：生活奢侈度 >> 公開職業收入 → 可用金錢切入
- **心理拉扯技巧**：用競爭者消費觸發損失厭惡，讓對方主動降姿態
- **榜單策略**：不在前幾名（競爭激烈）浪費資源，找中後段命中率更高
- **騎士團效應**：直播間只看不打賞的觀眾會將金主動向回報給主播

## 趣聞
旅遊期間直播主仍需對粉絲謊稱「在澳門工作很辛苦、腳很痠」——克里斯認為這種「在欺騙粉絲的同時被自己掌控」的禁忌感是最大成就感來源。
