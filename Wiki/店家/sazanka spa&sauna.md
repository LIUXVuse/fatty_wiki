# sazanka spa&sauna

**類型**：按摩（韓式搓澡 SPA）
**地點**：[[曼谷]]（[[泰國]]）
**服務類型**：韓式搓澡、SPA、額外加購服務

## 特色
曼谷的韓式搓澡 SPA，以 iPad 選妹 + 附餐點 + 韓式搓澡服務為特色，適合好友聚會式體驗。額外服務（脫衣/口服/全套）採加購模式，價格透明。

| 方案 | 費用 | 備註 |

| D 方案（基本）| 1380 THB（約 1200 TWD）| 含韓式搓澡 + 附餐點 |
| 加購：脫衣 | +500 THB | 視要求 |

## 服務特色
- **韓式搓澡**：正規去角質搓澡服務，附餐點
- **iPad 選妹**：用平板展示妹子照片，非現場選人
- **彈性加購**：基本套餐清晰，額外服務按需加購
- **適合聚會**：相比一般按摩店，附餐點的體驗感更完整
