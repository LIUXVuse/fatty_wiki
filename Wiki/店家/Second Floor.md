# Second Floor

**類型**：餐館／資訊站（非成人場所）
**地點**：[[天使城]]，[[菲律賓]]
**服務類型**：餐飲、社交、在地資訊交流

> [!warning] ⚠️ 類型更正
> stub 誤標「成人娛樂」。Second Floor 是天使城的菲律賓小餐館，提供中文歌，為台灣導遊與台灣訪客的聚集地，本身非成人場所。

## 特色
天使城（Angel City）中的台灣人友善餐館，是當地台灣導遊與老玩家的情報交流據點。嘉賓 AC浪子 在 S3EP230 特別推薦此處作為新手獲取在地資訊的第一站。

## 功能定位

| 功能 | 說明 |
| :--- | :--- |
| 台灣人聚集地 | 台灣導遊與老玩家常在此出沒 |
| 中文歌服務 | 提供中文歌曲，融入感強 |
| 資訊交換 | 可詢問最新店家狀況、Bar-fine 行情 |
| 用餐休息 | 一般餐館功能，可放鬆補給體力 |

## 實用技巧
- 新手入門：到 AC 的第一晚可先來 Second Floor 認識當地台灣人
- 直接問台灣導遊：店家推薦、妹子評測、安全資訊一次到位
- S3EP220 稱其為「Second Flow Bar」（同一地點，異名）
