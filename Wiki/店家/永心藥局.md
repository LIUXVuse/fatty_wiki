# 永心藥局

**類型**：中西藥局（非成人場所）
**地點**：[[台中]]南屯，[[台灣]]
**服務類型**：壯陽保養藥品＋合格藥師私密諮詢

## 特色
台中南屯的中西藥局，在節目聽眾間享有口碑。提供**有合格藥師的私密諮詢**服務，適合需要詢問壯陽藥使用方式但感到害羞的男性。

**聽眾優惠**：報「肥宅老司機」可獲得優惠（S3EP240 提及）。

## 諮詢流程
1. 進入藥局後告知**女店員**：「我要找藥師諮詢」
2. 藥師會引導至**私密處**（非公開空間）進行交談
3. 可安心詢問劑量、副作用、禁忌藥物搭配等問題

## 主要服務

| 分類 | 說明 |
| :--- | :--- |
| 保養組合 | 日常男性保養（如金胺酸、西地式低劑量） |
| 戰鬥用藥 | 臨陣前服用的壯陽藥（如威爾剛/西地式） |
| 中藥品項 | 中西合璧，多元選擇 |

## 常見諮詢項目（參考 S3EP257）

| 藥品 | 說明 |
| :--- | :--- |
| [[西地式]]（Cialis） | 副作用較小，5mg 可改善前列腺肥大 |
| [[威爾剛]]（Viagra） | 效果最強，副作用較大（頭痛/鼻塞），建議初試從 50mg |
| [[金胺酸]] | 保養用，早上空腹服用，微擴張血管 |

**重要禁忌**：若有服用高血壓藥，必須與西地式錯開時間（血壓藥早上，西地式晚上），否則有心血管風險。

## 注意事項
- 壯陽藥服用請搭配藥師建議，有高血壓藥者需錯開時間
- 絕對不要在網路購買來源不明的藥物（副作用與劑量未知，高風險）
- 此店為藥局，非成人娛樂場所
