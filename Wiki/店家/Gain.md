# Gain

**類型**：按摩店（Model 等級）
**地點**：[[首爾]]，[[韓國]]
**服務類型**：成人按摩，水床服務

## 特色
首爾的按摩店，定位為 Model 等級服務（價格與 [[道]] 相近），但實際表現讓嘉賓康熙大失所望。

嘉賓康熙在 S3EP200 中明確表示：
> 「不建議嘗試，直接去道即可。」

## 消費資訊
| 項目 | 費用 | 備註 |
| :--- | :--- | :--- |
| Model 等級 | 約 34 萬韓幣 | 與道相近 |

## 評價對比（Gain vs 道）

| 項目 | Gain | [[道]] |
| :--- | :--- | :--- |
| 費用 | ~34 萬韓幣 | 27-30 萬韓幣 |
| 妹子顏值 | 較低，年齡層較高 | 極高（平均 80 分以上） |
| 服務水準 | 較低 | 緊湊、專業 |
| 水床服務 | 相似技術水準 | 同上 |
| 整體推薦 | ❌ 不推薦 | ✅ 推薦 |

## 注意事項
- Gain 的水床服務技術與道相似（韓國店家可能有統一培訓標準）
- 但妹子年齡層較高，顏值落差大，整體體驗遠不如道
- 若有心儀首爾按摩店，建議直接鎖定[[道]]，不必多嘗試其他店家
