# 泰浴（Thai Bath House）

## 定義
泰國的傳統服務行業，位於紅燈區。遊客可以選擇妓女後進行一系列服務，通常包括浴缸洗浴、水床服務和床上服務。

## 操作流程
根據 [[2025年5月芭提雅探店指南]]：
1. 進店 → 找經理拿價格牌
2. 看妞 → 選妞
3. 前台付全款 → 給經理 100 銖小費
4. 上房 → 買飲料 → 脫衣
5. 浴缸放水 → 水床服務 → 浴缸服務 → 床上服務
6. 完事再洗 → 穿衣 → 給技師 100+ 銖小費
