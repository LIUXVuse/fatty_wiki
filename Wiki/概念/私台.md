# 私台

**類型**：術語 / 酒店玩法
**別名**：私檯、坐台、坐檯（逐字稿常見誤字：師台、師臺）

## 定義
私台（私檯）是台灣 KTV 酒店的一種坐台制度，指妹子固定陪同一桌客人，**不輪流轉檯**，從客人進來一直陪到離開。相對的是「公台」（妹子輪流換，一波波不同的進來）。

## 私台 vs 公台比較

| | 私台（私檯） | 公台 |
|---|---|---|
| 玩法 | 妹子固定，全程陪你到底 | 妹子輪流換，一波波不同的進來 |
| 計費 | 固定時段收費（如 1 小時 300 TWD） | 按輪或按時間計費 |
| 適合 | 想培養關係、聊天、建立熟客關係 | 想看多種妹子、花樣多 |
| 常見地區 | 彰化（越南妹私台店）、中南部 | 台中、台北部分店家 |

## 彰化越南私台店模式（[[大口袋]]分享，S3EP264）
- 費用：1 小時 300 TWD（全歸妹子，老闆賺包廂費和酒錢）
- 最少做 2 小時，兩小時 600 TWD（不含酒和秀武）
- 秀武（脫衣熱舞）：每首歌 200 TWD，妹子或客人都可點
- 妹子構成：外配姐姐（30-40 歲，老公上班去打工賺外快）+ 越南留學生年輕妹
- 姐姐尺度大（可手工 1,000 TWD）；年輕妹保守，看緣分
- **第一次去的攻略**：老闆只能排姐姐 → 請姐姐帶年輕妹進來訪客 → 給小費當場要 LINE → 下次直接私聯
