# 0.3 / 0.5 / 1（服務等級系統）

**類型**：術語
**別名**：0.3、0.5、1.0、1

## 出現集數
- [[肥宅老司機-S3EP4]] — 指服務的量級或次數（半套或全套）。
- [[肥宅老司機-S3EP34]] — 不含全套的半套或手技按摩服務。
- [[肥宅老司機-S3EP39]] — 半套或不含最後階段的服務。
- [[肥宅老司機-S3EP49]] — 僅提供按摩與手技服務，不含全套。
- [[肥宅老司機-S3EP74]] — 手槍服務（1/3 服務）；全套（1.0）為完整插入。
- [[肥宅老司機-S3EP75]] — 0.3 手槍服務；0.5 半套服務。
- [[肥宅老司機-S3EP92]] — 手淫（打手槍）服務。
- [[肥宅老司機-S3EP100]] — 特定服務等級。
- [[肥宅老司機-S3EP128]] — 業界對服務內容（如口交、全套等）的代碼稱呼。
- [[肥宅老司機-S3EP205]] — 半套服務（0.5）。
- [[肥宅老司機-S3EP213]] — 用手進行的服務（0.3）。
- [[肥宅老司機-S3EP235]] — 服務到 3 分（口交等）/ 服務到 5 分（特定限制）。

## 定義
服務尺度等級系統，數字越大代表服務內容越多：
- **0.3**：手部服務（手交/手槍），俗稱「點三」
- **0.5**：半套（口交或半套服務），俗稱「點五」
- **1 / 1.0**：全套（含本番），完整插入服務

## 相關概念
- [[全套]] — 1.0 的完整說法
- [[半套]] — 0.5 的完整說法
- [[點三店]] — 只做 0.3 的店家
- [[點五店]] — 只做 0.5 的店家
