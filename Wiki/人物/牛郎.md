# 牛郎

**類型**：術語 / 職業（男性陪酒從業者）
**相關概念**：[[應酬文化]]

## 定義
牛郎（男公關）是提供女性顧客陪伴、聊天、喝酒等情感服務的男性從業者。核心價值不在外貌，而在幽默感、酒量與帶動氣氛的能力（跳舞、魔術等）。

## 行業生態（S3EP97，[[基德]]分享）

[[基德]]（KID）揭露年輕時曾擔任男公關的親身經歷：

**客群**：經濟獨立或婚姻不順的熟女

**工作文化**：
- 女性顧客灌酒文化兇殘——隨意混酒，目標是把男公關灌倒
- 公司嚴禁與顧客發生性行為，違者開除（避免顧客丈夫來鬧事）

**行業規律**：
| 項目 | 內容 |
|------|------|
| 核心能力 | 幽默感、酒量、跳舞/魔術等帶動氣氛技能 |
| 巔峰年齡 | 20–25 歲 |
| 職業壽命 | 受限於兵役，壽命極短 |
| 營業高峰 | 凌晨 3 點後 |
