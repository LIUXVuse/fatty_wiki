# fatty_wiki — 個人 AI 知識庫

基於 Andrej Karpathy 的方法建立。用 Claude Code 自動整理文章、連結概念。

## 快速開始

1. **丟文章** → 把文章(.md / .txt / .pdf)丟到 `Raw/` 資料夾
2. **處理** → 在 Claude Code 說：`ingest` 或 `START`
3. **問問題** → 直接在 Claude Code 問任何問題
4. **用 Obsidian 看圖譜** → 把 `fatty_wiki` 資料夾用 Obsidian 開啟

## 指令速查

| 說這個 | 效果 |
|--------|------|
| `START` | 完整處理流程（新逐字稿 → Wiki） |
| `ingest` | 處理 Raw/ 裡的新文章 |
| `ask [問題]` | 從知識庫回答問題 |
| `health` | 健康檢查，找知識缺口 |
| `gap` | 自動補充缺口 |
| `fix` | 補救歷史資料缺口 |

## 資料夾說明

```
fatty_wiki/
├── Raw/              ← 把原始文章丟這裡（不進 git）
├── Wiki/
│   ├── 索引.md       ← 知識庫目錄（自動維護）
│   ├── 人物/         ← 投稿者、來賓、主持人（341 人）
│   ├── 店家/         ← 各類場所（529 家）
│   ├── 地點/         ← 城市、地區、街道（47 個）
│   ├── 概念/         ← 術語、行話（705 個）
│   ├── 來源/         ← 集數摘要、PDF 分析（286 篇）
│   └── Log.md        ← 操作記錄
├── tools/            ← 自動化處理腳本
└── CLAUDE.md         ← AI 指令手冊
```
